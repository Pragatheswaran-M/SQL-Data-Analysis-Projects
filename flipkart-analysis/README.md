# Flipkart Customer Behavior Analysis

## Problem Statement
Analyze customer purchase behavior to identify high-value customers, top categories, and spending patterns.

## SQL Concepts Used
- Joins
- CTEs
- Window Functions (LAG, LEAD, DENSE_RANK)

## Key Questions Solved
- Top customers by revenue
- Category-wise spending
- Order-to-order spending changes
- Growth vs decline customers

## Key Insights
- Top customers contribute majority of revenue
- Some customers show declining spending patterns
- Category-wise analysis helps target marketing
